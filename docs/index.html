Install

Three ways to get videopilot on your machine.

Step 1 — Install the package

pip install videopilot

FFmpeg must also be on PATH. See videopilot doctor to verify.

Step 2 — Wire it into your agent

Then wire it into your MCP client (e.g. ~/.copilot/mcp-config.json):

{
  "mcpServers": {
    "videopilot": {
      "type": "stdio",
      "command": "videopilot-mcp",
      "tools": ["*"]
    }
  }
}

After your client restarts, the agent can call all 20 videopilot.* tools.

Clone and install editable

git clone https://github.com/mbahgatTech/videopilot.gitcd videopilotpip install -e .

20 tools, one server.

Every step of the pipeline is exposed as an MCP tool an agent can call, schema-introspect, and idempotency-probe.

doctor
Verify ffmpeg, edge-tts, and whisper are on PATH.
voices
List available neural TTS voices.
list_projects
Enumerate projects under projects/.
project_status
Pipeline status for one project: state files, intermediates, final.
init
Create a new project scaffold.
import_source
Add another source video to an existing project.
read_state
Read one of the per-project JSON state files.
write_state
Write script / cut-plan / compose-plan JSON.
tts
Synthesise voiceover MP3s from script.json.
transcribe
Word-level transcript JSON + SRT via faster-whisper.
silence
Run silencedetect and emit a cut-plan candidate.
cut
Cut clips per cut-plan.json.
compose
Render final video per compose-plan.json.
export
Emit EDL / FCPXML / replay script for the timeline.
schema
JSON schemas for every state file the engine reads.
add_vo_segment
Append a voiceover segment to script.json.
add_slide
Append a slide to compose-plan.json.
set_compose_output
Patch the output block (codec, fps, resolution…).
preview_slide
Render one timeline item to out/preview-NNN.mp4.
is_up_to_date
Check whether downstream artifacts are still consistent.

Quick Start

A nine-step manual run, mirroring the README — every step is also an MCP tool an agent can call autonomously.

  1. Verify your environment
    Run videopilot doctor to confirm ffmpeg, edge-tts, and whisper are reachable.
  2. Create a project
    videopilot init my-clip scaffolds projects/my-clip/ with empty state files.
  3. Import a source video (optional)
    videopilot import-source my-clip path/to/raw.mp4.
  4. Author the voiceover script
    Edit script.json directly or call add_vo_segment from your agent.
  5. Synthesise voiceovers
    videopilot tts my-clip generates one MP3 per script segment.
  6. Plan your cuts
    Run videopilot silence my-clip raw1 to seed a cut-plan.json, then refine.
  7. Cut the clips
    videopilot cut my-clip writes trimmed clips into clips/.
  8. Build the timeline
    Author compose-plan.json (or call add_slide) and preview with videopilot preview-slide my-clip 0.
  9. Render the final MP4
    videopilot compose my-clip. Optional: videopilot export my-clip --fcpxml to hand off to an NLE.
